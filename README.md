# Sandbox 

### Purpose
-------

This repository is a development sandbox and playground for hosting apps, experimental repositories, and agent-based coding exercises. It is designed as a collaborative workspace for me, [Dr. Suyash Naik](https://suyashnaik.com) and AI agents to prototype ideas, learn agent-driven development patterns, and iterate rapidly.

### Use cases
---------

- Host multiple micro-projects, experiments, and prototype apps together.
- Agent-based coding learning space: create, test, and refine AI agents that assist with development tasks.
- Play together with AI agents: run multi-agent workflows, orchestration patterns, and human-in-the-loop experiments.
- Short-term iteration area for proofs-of-concept, demos, and learning exercises.
- Hack the boring stuff into easy tasks
